# Usage
<!-- usage -->
```sh-session
$ npm install -g forcoder
$ coder COMMAND
running command...
$ coder (-v|--version|version)
forcoder/0.0.0 darwin-x64 node-v11.10.0
$ coder --help [COMMAND]
USAGE
  $ coder COMMAND
...
```
